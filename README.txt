Game can be run using main.exe.
Code can also be compiled from source given PyGame, Esper, PyTMX, NumPy and SciPy are installed.
The project was created in Python 3.9.1.

The controls are as follows:

		Keyboard	XBoX 360 Controller
Steering	A/D		Left Thumbstick
Throttle	W		Right Trigger
Brake		S		Left Trigger
Gear up		K		Right Bumper
Gear down	M		Left Bumper
Clutch		P		A
Handbrake	Space		B
Back to menu	Esc		Start
